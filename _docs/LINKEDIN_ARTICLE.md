## The Tech Stack

Built with modern, proven technologies:

- **Electron** - For native Windows app experience
- **Alpine.js** - Lightweight, reactive UI (no React bloat)
- **Marked.js** - Fast markdown parsing
- **Highlight.js** - Beautiful syntax highlighting

Why this stack? Because it delivers a native-feeling app with minimal overhead. The entire app is fast, responsive, and feels right at home on Windows.

## Why I Built This

This wasn't about building the next unicorn startup. It was about solving a problem I experienced daily and creating something useful.

As developers, we often reach for heavy tools when we need something simple. VS Code is amazing, but sometimes you just want to read a file. Browser extensions work, but they're clunky and require Chrome to be open.

MarkVue fills that gap. It's the tool I wish existed when I first started working with AI-generated markdown files.
